# Online NGO Management System

## Overview

The Online NGO Management System is a web-based application developed using ASP.NET Web Forms, C#, and MySQL. The system provides a centralized platform for NGO verification, donor management, donation tracking, volunteer registration, and administrative monitoring. It helps improve transparency, accountability, and efficiency in managing NGO operations and donation activities.

## Features

### NGO Module

* NGO Registration
* NGO Profile Management
* NGO Verification Request Submission
* View Verification Status

### Donor Module

* Donor Registration and Login
* Manage Donor Profile
* Browse Verified NGOs
* Make Donations
* View Donation History

### Admin Module

* Secure Administrator Login
* NGO Verification and Approval
* Manage NGOs and Donors
* Monitor Donations
* Generate Reports
* System Management

### Donation Management

* Donation Tracking
* Donation Records Management
* Transaction Monitoring
* Donation History

### Security Features

* User Authentication
* Role-Based Access Control
* Secure Data Management
* Validation and Verification

## Installation

### Prerequisites

* Visual Studio 2022
* .NET Framework
* MySQL Server
* MySQL Connector for .NET

### Setup Steps

1. Clone the repository.

```bash
git clone https://github.com/omspatil/Online-NGO-Management-System.git
```

2. Open the solution file in Visual Studio.

3. Create the MySQL database.

4. Import the database script.

5. Update the database connection string in `Web.config`.

6. Build and run the project.

## Project Description

Developed a web-based Online NGO Management System using ASP.NET Web Forms, C#, and MySQL. The system allows NGOs to register and undergo administrator verification before receiving donations. It includes modules for donor management, donation tracking, volunteer registration, and admin monitoring, ensuring secure, transparent, and efficient management of NGO operations and donation activities.
